## Cloud Run

## 主な使用ライブラリ
- [Spring Boot (フレームワーク)](https://spring.io/projects/spring-boot)
- [DOMA2 (O/Rマッパー)](https://doma.readthedocs.io/en/latest/)


### IntelliJの設定

- DOMA plugin のインストール
    - メニューから`File` `Settings` `Plugins` を選択
    - `doma` を入力し、`Search Results` から `Doma Support` を選択し、インストール。
